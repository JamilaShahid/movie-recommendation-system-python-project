# 🎬 Movie Recommendation System

A Python-based Movie Recommendation System using Machine Learning and Streamlit.

## 🌐 Live Demo
[Click here to try the app!](https://movie-recommendation-system-python-project-a6p23ojypsgazokxb5h.streamlit.app)

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn (TF-IDF & Cosine Similarity)
- Pandas

## ⚙️ How it works
1. User enters a movie name
2. System finds similar movies using TF-IDF and Cosine Similarity
3. Top recommendations are displayed
